# Security Policy

## Supported Versions

This project is pre-1.0. Only the latest `0.x` release line receives security
fixes. Older pre-release snapshots are not maintained.

| Version | Supported          |
| ------- | ------------------ |
| 0.x     | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities privately — do **not** open a public
issue for a suspected vulnerability.

Email **tamircohen2468@gmail.com** with:

- A description of the vulnerability and its impact
- Steps to reproduce (proof-of-concept if available)
- Affected version(s) and environment

### Response SLA

| Stage                 | Target                       |
| --------------------- | ---------------------------- |
| Acknowledgement       | within 48 hours              |
| Initial assessment    | within 5 business days       |
| Fix or mitigation plan| within 30 days of triage     |

We will keep you informed throughout the process and credit you in the release
notes once a fix ships, unless you prefer to remain anonymous.
